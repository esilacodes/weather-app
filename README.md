# 🌍 Weather App

A simple weather desktop app built with Python and CustomTkinter.

## Features
- Current temperature and feels like
- Weather status, humidity and wind speed
- Dark theme
- Search with Enter key

## Requirements
- Python 3
- customtkinter
- requests
- python-dotenv

## Setup

1. Clone the repo:
   git clone https://github.com/esilacodes/weather-app.git

2. Install dependencies:
   pip install customtkinter requests python-dotenv

3. Create a .env file and add your API key:
   API_KEY=your_api_key_here

4. Run the app:
   python main.py

## API
Free API key from openweathermap.org
